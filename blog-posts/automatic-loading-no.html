the javascript part

so, how can the DOM be manipulated with javascript?
javascript can:

  • change all the elements and attributes in a page
  • change every CSS style in a page
  • remove existing elements and attributes from the page
  • add new elements and attributes (this is important in this situation)
  • react to any HTML event in the page
  • and also create those events

my previous method was changing elements and attributes of HTML elements.
What I should be doing is adding new elements and attributes. it seems a bit obvious now
an example:


js-css > global.js 

let newElement = document.createElement("p");
newElement.textContent = "Start running.";
document.body.appendChild(newElement);

// this will add a paragraph element with contents "Start running." to the document body.
