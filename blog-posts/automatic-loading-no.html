3-29-2025

i'm not adding automatic loading to my website anymore

i don't want to add automatic loading to my website anymore
what i mean by automatic loading is loading stuff like the footer, navigation, and banner with javascript to reduce copy and pasting
and i did that on my last website
i won't here though

why?

because i still had to copy and paste anyway
it doesn't just add the navigation, footer and banner to the page
instead, it finds a placeholder html element and then puts stuff inside
i still need to copy and paste that placeholder, so it removes the entire purpose
