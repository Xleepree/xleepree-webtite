pwamaker

pwamaker is a CLI tool for making PWAs (Progressive Web Apps) quickly on Windows, macOS, and Linux. It is built using Rust and the PWAs are built using wry. It builds the source for a PWA and puts it in someone's downloads folder. In addition, it optionally creates binaries like .exe for Windows, .dmg for macOS, and .deb/.AppImage/.tar.xz for Linux, so it can be used as a real app. (NOTE: this functionality has not been released yet)
